# Satinder Sidhu - Professional Portfolio

A modern, responsive single-page portfolio website showcasing professional experience, certifications, and projects.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Single Page Application**: Easy navigation with smooth scrolling
- **GitHub Integration**: Displays repositories and GitHub profile information
- **LinkedIn Integration**: Showcases work experience and certifications
- **Interactive Elements**: Hover effects, scroll animations, and dynamic navigation

## Sections

1. **Hero Section**: Introduction with profile photo, social links, and call-to-action buttons
2. **About**: Professional summary with key statistics
3. **Experience**: Timeline of work history and major projects
4. **Certifications**: Showcase of AWS, Google Cloud, and Microsoft certifications
5. **Projects**: Featured GitHub repositories with descriptions
6. **Contact**: Contact information and professional links

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter)
- SVG Icons

## How to Use

1. Open `index.html` in your web browser
2. For local development, you can use any local server:
   - Python: `python -m http.server 8000`
   - Node.js: `npx serve`
   - VS Code: Use Live Server extension

## Deployment Options

### GitHub Pages
1. Create a new repository on GitHub
2. Push these files to the repository
3. Go to Settings > Pages
4. Select the main branch as source
5. Your site will be live at `https://[username].github.io/[repository-name]`

### Netlify
1. Sign up at [Netlify](https://www.netlify.com/)
2. Drag and drop the project folder
3. Your site will be live instantly

### Vercel
1. Sign up at [Vercel](https://vercel.com/)
2. Import the repository
3. Deploy with one click

## Customization

To customize the portfolio:

1. **Colors**: Edit CSS variables in `styles.css` (lines 1-17)
2. **Content**: Update information in `index.html`
3. **Profile Photo**: Replace the GitHub avatar URL with your preferred image
4. **Projects**: Add or modify projects in the projects section
5. **Certifications**: Update certification details as needed
